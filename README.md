Project : Case01
Description : Testing https://demoqa.com/automation-practice-form dengan Automation Katalon



Cara Menjalankan:
1. Lakukan Clone ke Local
2. Pastikan sudah menginstall Katalon, yang memenuhi minimum requirement
3. Buka Katalon
4. Pastikan sudah melakukan login Katalon

A. Running Automation
1. Lakukan Open Project, untuk AutomationPracticeForm
2. Lakukan Akses Test Suite berikut pada folder Test Suites:
    - TSPracticeFormNumericalOnly
    - TSPracticeFormNegativeMandatory
    - TSPracticeFormInvalidFormatEmail
    - TSPracticeFormMinLengthMobileNumber
    - TSPracticeFormPositive
3. Lakukan run dengan klik icon run
4. Pilih Web Browser yang available
5. Pastikan semua tahapan pengujian berjalan sukses
6. Report xml akan otomatis tergenerate pada folder Reports

B. Generate Report PDF 
1. Pada folder report di Katalon studio, akses folder hasil report hingga menemukan file html
2. Klik kanan pada file
3. Klik Export As
4. Pilih PDF
5. Tentukan Lokasi folder
6. Berhasil report PDF



Requirements:
- Windows : 11
- Katalon : 10.4.2
